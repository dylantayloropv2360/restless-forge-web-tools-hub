## What Restless Forge Is

Restless Forge packages a set of small web utilities into a single browser-first hub. Every tool executes locally in the user session rather than routing data through a server, so the experience stays self-contained and works well as a static site.

It is aimed at anyone who wants fast access to everyday helpers without creating an account or jumping between separate apps. The current release centers on practical utilities like an hourly wage calculator, a hologram GIF generator, and a sand table pattern converter, giving it value for both casual use and lightweight creative work.

## Getting Started

Restless Forge can be used like a static web project or launched through a local development environment.

1. Clone the repository:
   git clone https://github.com/dylantayloropv2360/restless-forge-web-tools-hub.git
2. Move into the project folder:
   cd restless-forge-web-tools-hub
3. Install dependencies and start the dev server:
   npm install
   npm run dev

If you are not working locally, build the site and publish the generated static output to any hosting service that supports browser access.

## FAQ

**Does Restless Forge need a backend?**  
No. The tools are built to run entirely in the browser.

**Do I need an account to use it?**  
No sign-ups are required.

**How do I update the project?**  
Pull the latest repository changes, then rebuild and redeploy the static output if you are hosting it yourself.

**Where are settings stored?**  
Most behavior is defined in the client-side source and build configuration rather than on a remote server.

**What if a tool does not load correctly?**  
Try refreshing the browser, clear local site data if needed, and make sure your browser supports the required web features.
